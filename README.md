# Healthcare Appointment Platform

## Project Overview

Healthcare Appointment Platform is a full-stack web application inspired by modern healthcare booking systems such as Practo.

The platform enables patients to search for doctors, book appointments, manage schedules, and receive notifications through a secure and scalable microservices architecture.

---

## Features

### Patient Features
- User Registration & Login
- JWT Authentication
- Search Doctors
- Book Appointments
- Cancel Appointments
- View Appointment History

### Doctor Features
- Doctor Registration
- Profile Management
- Manage Availability
- View Scheduled Appointments

### Admin Features
- Manage Doctors
- Manage Patients
- Monitor Appointments
